2.Создать Spring Boot приложение.

Приложение делает подсчёт количества вызовов ресурса /counter (POST).

После перезапуска, приложение должно продолжать подсчёт с последней цифы (не с нуля).

При запуске /counter (GET) – показывать итоговое количество запросов.

При запуске /report (GET) вывести весь список запросов (список с временами запуска)
